## 如何直接部署在你的电脑上
1. 安装 [Node.js](https://nodejs.org/en/download/)
2. 把这个仓库clone到随便一个地方
3. cd *那个地方*
4. ```npm ci```
5. ```npm run serve``` (如果你需要调试，可以使用```dev```)

在 ```package.json``` 里面你可以找到大部分的运行配置，和我用的基本没什么差别，虽然但是大部分我也没怎么用过

## How to deploy on your computer
1. Install [Node.js](https://nodejs.org/en/download/)
2. Clone this repo to anywhere you want
3. ```cd``` to the folder (In command line of course)
4. run ```npm ci```
5. run ```npm run serve``` (For usage of debugging or development, run ```npm run dev``` instead)

You can find most of the configuration in ```package.json```,
most of them are automatically generated by electron-vite plugin, and idk if they can
work properly.

## If you want to build
Of course you can run ```npm run build:win``` to build for Windows.
And for other platforms, you can run commands like ```npm run build:mac```,
*but I haven't used the commands for platforms besides Windows, so give it a try!*

Sometimes you may find errors in the building process (not during vue-tsc), you can
retry it in command line with administrator privilege.
(I haven't met other situations, but this can solve most of the problems.)
